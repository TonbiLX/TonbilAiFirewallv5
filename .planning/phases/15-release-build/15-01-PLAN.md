  <action>
Bu gorev release APK uretmek icin gereken tum altyapiyi kurar:

1. **Keystore olustur** — `keytool` ile RSA 2048-bit, 10000 gun gecerli:
   ```
   keytool -genkey -v -keystore android/tonbilaios.keystore -keyalg RSA -keysize 2048 -validity 10000 -alias tonbilaios
   ```
   - CN: TonbilAiOS, O: Tonbil, L: Istanbul, C: TR
   - Sifre: `TonbilAiOS2026` (hem storePassword hem keyPassword ayni)

2. **keystore.properties olustur** (`android/keystore.properties`):
   ```properties
   storeFile=../tonbilaios.keystore
   storePassword=TonbilAiOS2026
   keyAlias=tonbilaios
   keyPassword=TonbilAiOS2026
   ```

3. **android/.gitignore guncelle** — su satirlari ekle:
   ```
   *.keystore
   *.jks
   keystore.properties
   ```

4. **proguard-rules.pro olustur** (`android/app/proguard-rules.pro`) — Research'teki kurallari aynen kullan:
   - Ktor: `-keep class io.ktor.** { *; }` + `-dontwarn io.ktor.**`
   - OkHttp + Okio: keep + dontwarn
   - Kotlinx Serialization: Companion + DTO siniflarini koru (`com.tonbil.aifirewall.data.remote.dto.**`)
   - Koin: keep + dontwarn
   - Glance, WorkManager, Security Crypto, Biometric: keep
   - Kotlin metadata: Signature, SourceFile, LineNumberTable, RuntimeVisibleAnnotations keepattributes

5. **build.gradle.kts guncelle** — signingConfigs + release buildType ekle:
   - Dosyanin basina `import java.util.Properties` ekle
   - `android {}` blogunun icine (defaultConfig'den sonra):
     ```kotlin
     val keystoreProperties = Properties()
     val keystorePropertiesFile = rootProject.file("keystore.properties")
     if (keystorePropertiesFile.exists()) {
         keystoreProperties.load(keystorePropertiesFile.inputStream())
     }

     signingConfigs {
         create("release") {
             storeFile = file(keystoreProperties["storeFile"] as? String ?: "")
             storePassword = keystoreProperties["storePassword"] as? String ?: ""
             keyAlias = keystoreProperties["keyAlias"] as? String ?: ""
             keyPassword = keystoreProperties["keyPassword"] as? String ?: ""
         }
     }

     buildTypes {
         release {
             isMinifyEnabled = true
             isShrinkResources = true
             proguardFiles(
                 getDefaultProguardFile("proguard-android-optimize.txt"),
                 "proguard-rules.pro"
             )
             signingConfig = signingConfigs.getByName("release")
         }
     }
     ```

6. **Release APK build et**:
   ```
   cd android && gradlew.bat assembleRelease
   ```
   Beklenen cikti: `android/app/build/outputs/apk/release/app-release.apk`

ONEMLI: `keystore.properties` ve `tonbilaios.keystore` git'e ASLA commit edilmemeli. `.gitignore` bunu saglar.
