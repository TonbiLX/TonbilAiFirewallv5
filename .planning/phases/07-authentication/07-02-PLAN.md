   - `fun onUsernameChange(value: String)` — username state guncelle
   - `fun onPasswordChange(value: String)` — password state guncelle
   - `fun login()` — viewModelScope.launch ile:
     1. isLoading = true, errorMessage = null

     4. Basarisiz: errorMessage = hata mesaji (Result.failure'dan), isLoading = false

   - `fun skipBiometric()` — biyometrigi atla, direkt Dashboard'a yonlendir
   - `fun clearError()` — errorMessage = null
