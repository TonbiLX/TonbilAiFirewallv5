1. util/HapticHelper.kt olustur:
   - object HapticHelper (singleton)
   - private var currentActivity: WeakReference<Activity>? = null
   - fun registerActivity(activity: Activity): currentActivity = WeakReference(activity)
   - fun unregisterActivity(): currentActivity = null

   - fun triggerHaptic(severity: String):
     * val activity = currentActivity?.get() ?: return (arka plandaysa tetikleme)
     * val view = activity.window.decorView
     * when(severity):
       - "critical" -> view.performHapticFeedback(HapticFeedbackConstants.REJECT)
       - "warning" -> view.performHapticFeedback(HapticFeedbackConstants.CONFIRM)
       - else -> return (info/low severity'de titresim yok)

   - IZIN GEREKMEZ: HapticFeedbackConstants ile View.performHapticFeedback() hic izin gerektirmez
   - Vibrator/VibrationEffect KULLANMA (izin gerektirir, karmasik, gereksiz)

2. TonbilApp.kt'yi guncelle:
   - registerActivityLifecycleCallbacks ekle (onCreate icinde, Koin'den sonra):
     ```kotlin
     registerActivityLifecycleCallbacks(object : ActivityLifecycleCallbacks {
         override fun onActivityResumed(activity: Activity) {
             HapticHelper.registerActivity(activity)
         }
         override fun onActivityPaused(activity: Activity) {
             HapticHelper.unregisterActivity()
         }
         // Diger callback'ler bos override
     })
     ```
   - observeSecurityEvents() icerisinde, NotificationHelper.showSecurityNotification'dan ONCE:
