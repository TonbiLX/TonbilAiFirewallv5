- Run `nft list chain bridge accounting upload 2>/dev/null` and capture output
- Run `nft list chain bridge accounting download 2>/dev/null` and capture output

- Check at least one counter line shows bytes > 0 (grep for `counter packets [1-9]` or `bytes [1-9]`)
- Print PASS if chains exist and have non-zero counters, WARN if chains exist but counters are zero (no traffic), FAIL if chains don't exist

**VALD-02: Conntrack ESTABLISHED:**
- Run `conntrack -L --state ESTABLISHED 2>/dev/null | wc -l`
- Check count > 0
- Print PASS with count, WARN if 0

**VALD-01: ARP verification (tcpdump):**

- Print the Pi MAC for reference
- Run `timeout 15 tcpdump -i eth0 -n -e arp -c 10 2>/dev/null` and capture output
- Extract all unique sender MAC addresses from the output
- Check if any non-Pi MACs appear
- Print PASS if all sender MACs equal Pi MAC, FAIL if foreign MACs detected
- Print all captured MACs for manual review
- Note: This check requires active LAN traffic. If no ARP packets captured in 15 seconds, print SKIP with explanation.

- Print PASS/FAIL for internet and DNS separately
