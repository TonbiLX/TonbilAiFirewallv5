   - Yeni metod ekle: `loadDdosAttackMap()` — lazy loading, sadece DDoS tab secildiginde cagrilacak:
     ```kotlin
     fun loadDdosAttackMap() {
         if (_uiState.value.ddosAttackMap != null) return // zaten yuklenmis
         viewModelScope.launch {
             securityRepository.getDdosAttackMap()
                 .onSuccess { map ->
                     _uiState.update { it.copy(ddosAttackMap = map) }
                 }
                 .onFailure { /* sessizce gec, mevcut DDoS tab verileri yeterli */ }
         }
     }
     ```
   - `refresh()` metodunda ddosAttackMap'i null'a resetle ki sonraki tab seciminde yeniden yuklensin:
