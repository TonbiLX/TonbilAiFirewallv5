ChatScreen.kt dosyasindaki `ChatBubble` composable'ini genislet:

1. Mevcut `Text(text = message.content, ...)` satirini bir icerik analiz blogu ile degistir. Sadece `role == "assistant"` mesajlar icin yapilandirilmis render yap; user mesajlari aynen kalsin.

2. Yeni private composable `AssistantMessageContent(content: String)` olustur:
   ```kotlin
   @Composable
   private fun AssistantMessageContent(content: String) {
       val trimmed = content.trimStart()
       when {
           trimmed.startsWith("{") || trimmed.startsWith("[") -> JsonContentBlock(trimmed)
           trimmed.contains("| ---") || (trimmed.contains("| ") && trimmed.count { it == '\n' } >= 2 && trimmed.lines().count { it.trimStart().startsWith("|") } >= 3) -> TableContentBlock(trimmed)
           else -> Text(content, color = TextPrimary, fontSize = 13.sp, lineHeight = 18.sp)
       }
   }
   ```

3. `JsonContentBlock(content: String)` composable olustur:
   - `try { Json.parseToJsonElement(content) }` ile parse et (kotlinx.serialization.json.Json)
   - JsonObject ise: key-value satirlari Column icinde goster — her key NeonCyan, deger TextPrimary, araya 0.5dp GlassBorder Divider
   - JsonArray ise: her eleman icin numarali liste (LazyColumn degil, Column — genellikle kisa)
   - Parse basarisiz ise: duz Text olarak goster (fallback)
   - Tum blogu GlassBg arka planli RoundedCornerShape(8.dp) Box icine al, 8.dp padding

4. `TableContentBlock(content: String)` composable olustur:
   - `|` ile baslamayan satirlari filtrele, `| --- |` ayrac satirini atla
   - Ilk satir header (NeonCyan + FontWeight.Bold), geri kalan satirlar veri (TextPrimary)
   - Row + weight ile sutunlari esit dagit
   - Tum tabloyu GlassBg Box icine al, horizontalScroll ile genis tablolari kaydirilabilir yap

5. ChatBubble icindeki `Column` blogu guncelle:
   ```kotlin
   // ONCE: Text(text = message.content, color = if (isUser) NeonCyan else TextPrimary, ...)

       Text(text = message.content, color = NeonCyan, fontSize = 13.sp, lineHeight = 18.sp)
   } else {
       AssistantMessageContent(content = message.content)
   }
   ```
